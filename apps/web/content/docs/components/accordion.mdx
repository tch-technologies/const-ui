---
title: Accordion
description: A vertically stacked set of interactive headings that each reveal an associated section of content.
---

<ComponentPreview 
  id="accordion-basic"
  code={`import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "@const-ui/react"

export function AccordionDemo() {
  return (
    <Accordion type="single" collapsible className="w-full">
      <AccordionItem value="item-1">
        <AccordionTrigger>Is it accessible?</AccordionTrigger>
        <AccordionContent>
          Yes. It adheres to the WAI-ARIA design pattern.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}`}
/>

## Props

### Accordion

| Prop | Type | Description |
| :--- | :--- | :--- |
| `type` | `"single" \| "multiple"` | Controls whether one or multiple items can be expanded. |
| `collapsible` | `boolean` | Allows all items to be collapsed when `type` is `single`. |
| `defaultValue` | `string` | The value of the item that should be open by default. |

## Structure

- `Card`: The wrapper component.
- `CardHeader`: Contains the title and description.
- `CardTitle`: A level 3 heading for the card title.
- `CardDescription`: Subtext positioned below title.
- `CardContent`: Main container for card body.
- `CardFooter`: Contains actions positioned at bottom.

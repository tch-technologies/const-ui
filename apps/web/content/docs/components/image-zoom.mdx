## Features

- **Accessible & Seamless**: Optimized transitions compatible with keyboard shortcuts (Esc key close).
- **Scroll-to-Dismiss**: Closes automatically when the user scrolls past the expanded portal.
- **Styled Overlays**: Native dark mode support via theme-aware backdrop filters.

## Props

### ImageZoom

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `zoomMargin` | `number` | `40` | Space between expanded image boundary and viewport edge. |
| `isZoomed` | `boolean` | - | Toggle state to create a fully controlled instance. |
| `onZoomChange` | `(value: boolean) => void` | - | Called instantly on trigger event state switch. |

## Props

### Select

Wraps Radix UI Select primitive.

| Prop | Type | Description |
| :--- | :--- | :--- |
| `value` | `string` | The controlled selection state. |
| `defaultValue` | `string` | The default selection value. |
| `onValueChange` | `(value: string) => void` | Event when state changes. |

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `checked` | `boolean` | - | Controlled boolean state. |
| `defaultChecked` | `boolean` | `false` | Initial value when uncontrolled. |
| `onCheckedChange` | `(checked: boolean) => void` | - | Event fired on state flip. |
| `disabled` | `boolean` | `false` | When true, interaction becomes inert. |

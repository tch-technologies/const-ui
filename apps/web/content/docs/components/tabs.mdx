## Curved Design

Utilize the `curved` variant to create a modern, professional tab set with inverted-radius corners that merge seamlessly into the container below (inspired by browser-tab styling).

## Props

### Tabs

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `defaultValue` | `string` | - | Initial active tab on mount. |
| `value` | `string` | - | Controlled active tab item. |
| `onValueChange` | `(value: string) => void` | - | Triggered when selected tab changes. |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Keyboard nav behavior. |

This is a library of accessible, high-performance React components. Designed with a clean aesthetic, accessibility first principles, and maximum developer productivity.

### Features

- **Accessible:** Fully supports WAI-ARIA patterns out of the box.
- **Tailwind CSS:** Styled with Tailwind utility classes for easy configuration.
- **Radix UI primitives:** Built on top of rock-solid headless primitives.
- **Dark Mode Ready:** Integrated runtime light/dark mode switching.

### Installation

Run the following command to install the package in your project:

```bash
pnpm install @thecodinghqs/ui
```
